# Student AI Usage Data Analysis and Wrangling
## Project Overview
This project is based on analyzing how Artificial Intelligence tools are influencing student learning and daily academic activities. The dataset contains information about students, their AI tool usage, daily usage hours, education level, purpose of using AI, impact on grades, and satisfaction levels.
The main aim of this project is to clean and organize raw data before analysis. Real-world datasets usually contain duplicate values, formatting issues, spelling mistakes, and inconsistent data. This project focuses on performing data preprocessing and wrangling to prepare a clean dataset for future analysis and visualization.

## Objectives
• Understand student AI usage behavior  
• Perform data cleaning and preprocessing  
• Remove duplicate records  
• Handle missing values  
• Standardize text formatting  
• Correct spelling inconsistencies  
• Convert data into analysis-ready format  
• Generate basic visual insights
